## 📦 Installation & Setup

To run any of my projects locally, follow these generic steps:

### 1️⃣ Clone the Repository

```sh

cd [project-name]
```

### 2️⃣ Install Dependencies

```sh

```

### 3️⃣ Run the Development Server

```sh
npm run dev
```
